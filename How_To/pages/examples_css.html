Seletor por tag

Seletor por tag.

Css:

span{
background-color: skyblue;
}

Html:

<span>Texto com tag "span".</span>
<p>Texto com tag "p".</p>

Resultado:

Texto com tag "span".
Texto com tag "p".

Seletor por classe

Seletor por classe.

Css:

.red{
color: #f33;
}
.yellow-bg{
background: #ffa;
}

Html:

<p>parágrafo com classe ".red".</p>
<p>parágrafo com classe ".yellow-bg".</p>
<p>parágrafo sem classe</p>

Resultado:

parágrafo com classe ".red".
parágrafo com classe ".yellow-bg".
parágrafo sem classe

Seletor Universal

Seletor Universal.

Css:

/*Seleciona todos os elementos*/
*{
background: skyblue;
color: blue;
}

Html:

<span>Texto com tag "span".</span>
<p>Texto com tag "p".</p>
<pre>Texto com tag "pre".</pre>

Resultado:

Texto com tag "span".
Texto com tag "p".
Texto com tag "pre".

Seletor por atributo

Seletor por atributo.

Css:

/* Links internos, começando com "#" */
a[href^="#"]{
background: skyblue;
color: red;
}

Html:

<a href="#top">TAG 'a' com atributo 'href="#top"'.</a>

Resultado:

TAG 'a' com atributo 'href="#top"'.
